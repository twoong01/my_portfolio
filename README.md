## 설명

노션, 템플릿 소스 등을 이용할 수 있었지만 `저만의 포트폴리오` 사이트를 가지고 싶어 제작했습니다.<br>
주요 색상인 `black & white`로 심플하고 차분한 느낌을 줬고,<br>
간단한 자기 소개와 프로젝트, 이력 등 중요도 순으로 배치해 내용을 구성했습니다.<br>
이를 통해 포트폴리오를 보는 사람들에게 원하는 정보를 쉽게 전달할 수 있도록 했습니다.<br><br>
